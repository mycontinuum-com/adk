# General advice

- Opt for simple, cautious code, which can be relied upon by millions of people who use the Anima platform.
- Only build what is necessary and avoid overbuilding.
- Always do a rigorous self-review of your code changes.

# Validation

Run ADK tests with `npm run test:adk` for improved time and memory efficiency

# Migration

The ADK regularly makes breaking changes, read MIGRATION.md after upgrading
